---
title: "How to delete a project in Upsolve"
description: "Get started with Project Operations: Learn how you can delete a project in Upsolve."
---

Follow these steps to delete a project in Upsolve:

1. **Access the project menu**: Start by identifying the project you wish to delete. Locate the project and click on the three-dots button (ellipsis icon) associated with it to open the project menu.

<img
  src="https://cdn.peaka.com/project-operations/click-3-dot.png"
  alt="open context menu of app view"
/>

2. **Select 'Delete'**: Once you've accessed the project menu, you'll find the 'Delete' option listed among other project actions. Click on 'Delete' to proceed with project removal.

<img
  src="https://cdn.peaka.com/project-operations/delete-app-1.png"
  alt="delete button in context menu view"
/>

3. **Confirm deletion**: After selecting 'Delete,' a confirmation popup will appear to ensure you intend to delete the project. Review the confirmation message carefully.

<img
  src="https://cdn.peaka.com/project-operations/delete-app-2.png"
  alt="delete project modal view"
/>

5. **Project removed**: Once confirmed, the project will be permanently removed from your account. You will no longer have access to it.

<img
  src="https://cdn.peaka.com/project-operations/delete-app-3.png"
  alt="delete project modal view"
/>

By following these steps, you can successfully delete a project in Upsolve.
