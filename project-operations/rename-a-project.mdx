---
title: "How to rename a project in Upsolve"
description: "Get started with Project Operations: Learn how you can change the name of your Upsolve project."
---

This guide will walk you through the process of renaming a project in Upsolve. Follow these simple steps to update the name of your project:

1. **Access the project menu**: To start renaming your project, find the project you want to update and click on the three-dots button (ellipsis icon) associated with the project. This action will open the project menu.

<img
  src="https://cdn.peaka.com/project-operations/click-3-dot.png"
  alt="open context menu of app view"
/>

2. **Select 'Rename'**: Within the project menu, locate and click the '**Rename**' option. This will initiate the renaming process.

<img
  src="https://cdn.peaka.com/project-operations/rename-app-1.png"
  alt="rename button in context menu view"
/>

3. **Edit the project name**: A popup window will appear, allowing you to edit the name of your project. Enter the new name you've chosen for your project in the provided field.

<img
  src="https://cdn.peaka.com/project-operations/rename-app-2.png"
  alt="rename project modal view"
/>

4. **Confirm the change**: After entering the new project name, click the '**Change**' button to confirm and apply the name change.

5. **Name successfully changed**: Your project's name will now be updated with the new name you provided. You'll see the new name displayed within the Upsolve interface.

<img
  src="https://cdn.peaka.com/project-operations/rename-app-3.png"
  alt="success renaming project view"
/>

By following these steps, you can easily rename a project in Upsolve.
