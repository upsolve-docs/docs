---
title: "How to clone a project in Upsolve"
description: "Get started with Project Operations: Learn how you can clone an existing project in Upsolve."
---

Follow these simple steps to clone an existing project in Upsolve:

1. **Access the project menu**: Start by identifying the project you want to clone. Locate the project, and you'll find a three-dots button (ellipsis icon) associated with it. Click on this button to access the project menu.

<img
  src="https://cdn.peaka.com/project-operations/click-3-dot.png"
  alt="open context menu of app view"
/>

2. **Select 'Clone'**: After clicking the three-dots button, a list of project actions will appear. From this list, select the 'Clone' option.

<img
  src="https://cdn.peaka.com/project-operations/clone-app-1.png"
  alt="clone button in context menu view"
/>

3. **Adjust the project name (optional)**: A popup window will appear, and Upsolve will automatically assign a name to the cloned project. If you prefer a different name, you can edit it at this stage.

<img
  src="https://cdn.peaka.com/project-operations/clone-app-2.png"
  alt="clone project modal view"
/>

4. **Clone the project**: To initiate the cloning process, simply click the "Clone" button within the popup. Upsolve will duplicate the project, and you will be able to access it from your project list.

<img
  src="https://cdn.peaka.com/project-operations/clone-app-3.png"
  alt="success cloning project view"
/>

By following these steps, you can easily clone an existing project in Upsolve.
