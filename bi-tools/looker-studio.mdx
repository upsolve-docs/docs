### Prerequisites:

- **A Google account**: To access **Looker Studio**, you will need a Google account. If you already have one, you can directly log into Looker Studio. If not, you can create an account in the [Looker Studio](https://lookerstudio.google.com/) page.

- A Upsolve account and a Upsolve Project.

- An API Key you created in your Upsolve Project.

## Using Upsolve as a Data Source in Looker Studio

1. Prepare your data in [Upsolve](https://upsolve.studio).

2. Click on the following link to navigate Looker Studio: <a href="https://lookerstudio.google.com" target="_blank">https://lookerstudio.google.com</a>

3. Click "Create" button on the left side of the page and then click "Data Source"

   <img
     src="https://cdn.peaka.com/bi-tools/looker-studio/looker-studio-create.png"
     style={{ border: "1px solid #e2e4e7" }}
   />

4. Search "Upsolve" in searchbar and select the "Upsolve" connector in "Partner Connectors" list to add Upsolve to Looker Studio as a data source.

   <img
     src="https://cdn.peaka.com/bi-tools/looker-studio/looker-studio-search.png"
     style={{ border: "1px solid #e2e4e7" }}
   />

5. To confirm the action, click on the AUTHORIZE button.

   <img
     src="https://cdn.peaka.com/bi-tools/looker-studio/looker-studio-1.png"
     style={{ border: "1px solid #e2e4e7" }}
   />

6. Sign in to Looker Studio using your Google account credentials and grant Google access rights by clicking on the ALLOW button.

7. Paste your Upsolve Project API Key to the "Key" field in Looker Studio.
   <img
     src="https://cdn.peaka.com/bi-tools/looker-studio/looker-studio-2.png"
     style={{ border: "1px solid #e2e4e7" }}
   />

<Warning>
  Upsolve's Looker Studio connector does not support Partner API Keys. Please
  make sure you enter an API Key that is created from a specific Upsolve
  Project.
</Warning>

8. Select a Catalog.

   <img
     src="https://cdn.peaka.com/bi-tools/looker-studio/looker-studio-3.png"
     style={{ border: "1px solid #e2e4e7" }}
   />
   <img
     src="https://cdn.peaka.com/bi-tools/looker-studio/looker-studio-4.png"
     style={{ border: "1px solid #e2e4e7" }}
   />

9. Select a Schema (if you are working with a Upsolve Table, a Upsolve Query, or a BI Table, you will proceed to select a Table without being asked to select a Schema).

   <img
     src="https://cdn.peaka.com/bi-tools/looker-studio/looker-studio-5.png"
     style={{ border: "1px solid #e2e4e7" }}
   />
   <img
     src="https://cdn.peaka.com/bi-tools/looker-studio/looker-studio-6.png"
     style={{ border: "1px solid #e2e4e7" }}
   />

10. Select a Table and click on the "Connect" button.

    <img
      src="https://cdn.peaka.com/bi-tools/looker-studio/looker-studio-7.png"
      style={{ border: "1px solid #e2e4e7" }}
    />
    <img
      src="https://cdn.peaka.com/bi-tools/looker-studio/looker-studio-8.png"
      style={{ border: "1px solid #e2e4e7" }}
    />

11. Browse the fields and rename the Data Source if needed.

    <img
      src="https://cdn.peaka.com/bi-tools/looker-studio/looker-studio-9.png"
      style={{ border: "1px solid #e2e4e7" }}
    />

12. Add the data source to your report.

You are all set now! You have successfully added Upsolve to Looker Studio as a Data Source.

You can now use your queries or tables to create insightful reports in Looker Studio.
