## Connecting Teradata to Upsolve

Now that you have your Teradata API Key, you can connect it to Upsolve using the following steps:

1. Navigate to the data model page in Upsolve.

2. In the side menu bar, click the "New Data Source" button to initiate the connection process.

3. In the modal that appears, select "Teradata" as your data source.

4. Enter a name for your data source name and paste values that you got before.

<img src="https://cdn.peaka.com/integrations/teradata-1.png" />

5. Click "Next" to create your Teradata data source.

Congratulations! You've successfully connected Teradata to Upsolve. You can now access and manage your Teradata data seamlessly within the Upsolve platform.
