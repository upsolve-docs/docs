This article guides you through the process of connecting Databricks to Upsolve, allowing you to seamlessly access your Databricks data within the Upsolve platform.

## Prerequisites

Before you start, you will need the following information from Databricks.

- URL
- PORT
- HTTP Path
- User
- Password

Follow the steps below to obtain it:

1. To obtain get URL, PORT and HTTP path, check this [article](https://docs.databricks.com/en/integrations/compute-details.html#:~:text=do%20the%20following%3A-,Log%20in%20to%20your%20Databricks%20workspace.,%2C%20Port%2C%20and%20HTTP%20path.).

2. To obtain get User and Password, check this [article](https://docs.databricks.com/api/account/accountusers/get).

## Connecting Databricks to Upsolve

Now that you have obtained the necessary integration data and credentials, you can connect Databricks to Upsolve using the following steps:

1. Navigate to the data model page in Upsolve.

2. In the side menu bar, click the "New Data Source" button to initiate the connection process.

3. In the modal that appears, select "Databricks" as your data source.

4. Enter a name for your data source, input the credentials you obtained in the previous steps.

<img src="https://cdn.peaka.com/integrations/databricks-1.png" />

5. Click "Next" to create your Databricks data source.

Congratulations! You've successfully connected Databricks to Upsolve. You can now access and manage your Databricks data seamlessly within the Upsolve platform.
