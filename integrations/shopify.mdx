This article guides you through the process of connecting Shopify to Upsolve, allowing you to seamlessly access your Shopify data within the Upsolve platform.

## Prerequisites

Before you start, you will need Admin API Access Token and Subdomain from Shopify.

Follow the steps below to obtain it:

1. Go to this [link](https://www.shopify.com/) and log in to your Shopify account.
2. Click the settings icon located in the lower-left corner of the website.
3. If you already have a Shopify app, select it. If not, create a new app.

![](https://cdn.peaka.com/integrations/shopify-1.png)

1. Click the "Develop apps" button, as shown in the image below.

![](https://cdn.peaka.com/integrations/shopify-2.png)

1. Configure your Admin API scopes and Storefront API scopes as needed for your integration.

![](https://cdn.peaka.com/integrations/shopify-3.png)

1. After configuring your API scopes, click "Install app" button to generate Admin API Access Token.

![](https://cdn.peaka.com/integrations/shopify-4.png)

1. Copy the Access Token generated for your app.

![](https://cdn.peaka.com/integrations/shopify-5.png)

1. Go to the "Domains" section on the side panel of the website to obtain or connect your Shopify subdomain.

![](https://cdn.peaka.com/integrations/shopify-6.png)

## Connecting Shopify to Upsolve

Now that you have your requirements, you can connect it to Upsolve using the following steps:

1. Navigate to the data model page in Upsolve.
2. In the side menu bar, click the "New Data Source" button to initiate the connection process.
3. In the modal that appears, select "Shopify" as your data source
4. Enter your data source name, Admin API Access Token and Subdomain.\
   ![Screenshot 2025-04-29 at 16.12.07.png](/Screenshot2025-04-29at16.12.07.png)
5. Click "Authorize" to create your Shopify data source.

Congratulations\! You've successfully created a data source between Shopify and Upsolve. You can now access and manage your Shopify data seamlessly within the Upsolve platform.
