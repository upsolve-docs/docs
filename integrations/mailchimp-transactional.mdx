This article guides you through the process of connecting Mailchimp Transactional to Upsolve, allowing you to seamlessly access your Mailchimp Transactional data within the Upsolve platform.

## Prerequisites

Before you start, you will need an API Key from Mailchimp Transactional.

Follow the steps below to obtain it:

1. Go to this [link](https://us21.admin.mailchimp.com/account/api/) to create or obtain your Mailchimp Transactional API Key.

2. Click the "Create a Key" button to generate your API Key.

<img src="https://cdn.peaka.com/integrations/transactional-integration-1.png" />

## Connecting Mailchimp Transactional to Upsolve

Now that you have your requirements, you can connect it to Upsolve using the following steps:

1. Navigate to the data model page in Upsolve.

2. In the side menu bar, click the "New Data Source" button to initiate the connection process.

3. In the modal that appears, select "Mailchimp Transactional" as your data source.

4. Enter your data source name and API Key.

<img src="https://cdn.peaka.com/integrations/transactional-integration-3.png" />

5. Click "Authorize" to create your Mailchimp Transactional data source.

Congratulations! You've successfully created a new data source between Mailchimp Transactional and Upsolve. You can now access and manage your Mailchimp Transactional data seamlessly within the Upsolve platform.
