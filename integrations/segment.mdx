This article guides you through the process of connecting Segment to Upsolve, allowing you to seamlessly access your Segment data within the Upsolve platform.

## Prerequisites

Before you start, you will need an API Key from Segment.

Follow the steps below to obtain it:

1. Navigate to the data model page in Upsolve.

2. In the side menu bar, click the "New Data Source" button to initiate the connection process.

3. In the modal that appears, select "Segment" as your data source.

4. Input the necessary information, including the Segment Catalog Name, and proceed by clicking the "Next" button.

<img src="https://cdn.peaka.com/integrations/segment-1.png" />

5.  Upon clicking the "Next" button, your unique API Key will be automatically generated. Please note that the API Key will only be displayed once.

<img src="https://cdn.peaka.com/integrations/segment-2.png" />

6.  Copy the generated API Key from Upsolve and paste it into the designated field within your Segment account settings.

## Adding Upsolve Destination in Segment

To integrate Upsolve with Segment, follow these steps to paste your API Key:

1. Navigate to the Upsolve Destination page on Segment by clicking on the following link:

https://app.segment.com/{your-workspace-name}/destinations/catalog/upsolve

2.  On the Upsolve Destination page, locate and click on the "Add Destination" button.

<img src="https://cdn.peaka.com/integrations/segment-3.png" />

3. In the dialogue box that appears, provide a name for your destination and proceed by clicking on the "Create Destination" button.

<img src="https://cdn.peaka.com/integrations/segment-4.png" />

4. Once the destination is created, locate the designated field for API Key entry. Paste the API Key obtained from Upsolve into this field.

<img src="https://cdn.peaka.com/integrations/segment-5.png" />

5. After pasting the API Key, make sure to save your changes to finalize the integration.
