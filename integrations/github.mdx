This article guides you through the process of connecting Github to Upsolve, allowing you to seamlessly access your Github data within the Upsolve platform.

## Prerequisites

Before you start, you will need the following information from Github.

- Personal Access Token
- Organization Name

Follow the steps below to obtain it:

1. To obtain get Personal Access Token, check this [article](https://docs.github.com/en/authentication/keeping-your-account-and-data-secure/managing-your-personal-access-tokens#creating-a-personal-access-token-classic).

2. To obtain get Organization Name, check this [article](https://docs.github.com/en/account-and-profile/setting-up-and-managing-your-personal-account-on-github/managing-your-membership-in-organizations/accessing-an-organization).

## Connecting Github to Upsolve

Now that you have obtained the necessary integration data and credentials, you can connect Github to Upsolve using the following steps:

1. Navigate to the data model page in Upsolve.

2. In the side menu bar, click the "New Data Source" button to initiate the connection process.

3. In the modal that appears, select "Github" as your data source.

4. Enter a name for your data source, input the credentials you obtained in the previous steps.

<img src="https://cdn.peaka.com/integrations/github-1.png" />

5. Click "Authorize" to create your Github data source.

Congratulations! You've successfully connected Github to Upsolve. You can now access and manage your Github data seamlessly within the Upsolve platform.
