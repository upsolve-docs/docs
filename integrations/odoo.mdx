This article guides you through the process of connecting Odoo to Upsolve, allowing you to seamlessly access your Odoo data within the Upsolve platform.

## Prerequisites

Before you start, you will need the following information from Odoo.

- Odoo URL
- Database Name
- User Name
- Password

Follow the steps below to obtain it:

1. Odoo URL is your project URL like https://example-app.odoo.com/

2. The Database Name is the specific name of the database you wish to connect. It can be found in the URL subdomain, such as `example-app` in the URL https://example-app.odoo.com/.

3. User Name is the username of the service account.

4. Password is the password for the service account user.

## Connecting Odoo to Upsolve

1. Navigate to the data model page in Upsolve.

2. In the side menu bar, click the "New Data Source" button to initiate the connection process.

3. In the modal that appears, select "Odoo" as your data source.

4. Enter a name for your data source and input credentials you obtained in the previous step.

5. Click "Next" to establish the data source.
