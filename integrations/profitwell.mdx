This article guides you through the process of connecting ProfitWell to Upsolve, allowing you to seamlessly access your ProfitWell data within the Upsolve platform.

## Prerequisites

Before you start, you will need an API Key from ProfitWell.

Follow the steps below to obtain it:

1. Go to this [link](https://www2.profitwell.com/app/account/integrations) to create or obtain your ProfitWell API Key or private token.

2. Copy your private token for use in the integration.

## Connecting ProfitWell to Upsolve

Now that you have your requirements, you can connect it to Upsolve using the following steps:

1. Navigate to the data model page in Upsolve.

2. In the side menu bar, click the "New Data Source" button to initiate the connection process.

3. In the modal that appears, select "ProfitWell" as your data source.

4. Enter your data source name and API Key.

<img src="https://cdn.peaka.com/integrations/profitwell-1.png" />

5. Click "Authorize" to create your ProfitWell data source.

Congratulations! You've successfully created a new data source between ProfitWell and Upsolve. You can now access and manage your ProfitWell data seamlessly within the Upsolve platform.
