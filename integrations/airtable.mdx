This article guides you through the process of connecting Airtable to Upsolve, allowing you to seamlessly access your Airtable data within the Upsolve platform.

## Prerequisites

Before you start, you will need an Personal Access Token from Airtable.

Follow the steps below to obtain it:

1. Visit the [Airtable website](https://airtable.com).

2. Click on the "Developer Hub" button, as shown in the image below:

<img src="https://cdn.peaka.com/integrations/airtable-integration-1.png" />

3. Create an API Token by clicking the "Create Token" button. You can select the necessary scopes and workspaces for your integration:

<img src="https://cdn.peaka.com/integrations/airtable-integration-2.png" />

4. Once the token is created, it will be displayed in a modal. Copy the API Token for later use:

<img src="https://cdn.peaka.com/integrations/airtable-integration-3.png" />

## Connecting Airtable to Upsolve

Now that you have your Airtable API Token, you can connect it to Upsolve using the following steps:

1. Navigate to the data model page in Upsolve.

2. In the side menu bar, click the "New Data Source" button to initiate the connection process.

3. In the modal that appears, select "Airtable" as your data source.

4. Enter a name for your data source name and paste the Airtable API Token into the "Personal Access Token" input field.

<img src="https://cdn.peaka.com/integrations/airtable-integration-4.png" />

5. Click "Authorize" to create your Airtable data source.

Congratulations! You've successfully connected Airtable to Upsolve. You can now access and manage your Airtable data seamlessly within the Upsolve platform.
