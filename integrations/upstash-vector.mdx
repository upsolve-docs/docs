This article guides you through the process of connecting Upstash Vector to Upsolve, allowing you to seamlessly access your Upstash Vector data within the Upsolve platform.

## Prerequisites

Before you start, you will need the following information from Upstash Vector.

- Index Endpoint
- Token

Follow the steps below to obtain it:

1. To obtain your Index Endpoint and Token, navigate to the project dashboard. If you don't have an existing Upstash Vector project, create a new one to access these credentials.

<img src="https://cdn.peaka.com/integrations/upstash-vector-1.png" />

## Connecting Upstash Vector to Upsolve

Now that you have obtained the necessary integration data and credentials, you can connect Upstash Vector to Upsolve using the following steps:

1. Navigate to the data model page in Upsolve.

2. In the side menu bar, click the "New Data Source" button to initiate the connection process.

3. In the modal that appears, select "Upstash Vector" as your data source.

4. Enter a name for your data source and paste values that you got before.

<img src="https://cdn.peaka.com/integrations/upstash-vector-2.png" />

5. Click "Authorize" to create your Upstash Vector data source.

Congratulations! You've successfully connected Upstash Vector to Upsolve. You can now access and manage your Upstash Vector data seamlessly within the Upsolve platform.
