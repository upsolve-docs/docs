This article guides you through the process of connecting Qdrant to Upsolve, allowing you to seamlessly access your Qdrant data within the Upsolve platform.

## Prerequisites

Before you start, you will need the following information from Qdrant.

- URL
- API Key

Follow the steps below to obtain it:

1. To obtain URL and API Key, check this [article](https://qdrant.tech/documentation/cloud/authentication/#).

## Connecting Qdrant to Upsolve

Now that you have obtained the necessary integration data and credentials, you can connect Qdrant to Upsolve using the following steps:

1. Navigate to the data model page in Upsolve.

2. In the side menu bar, click the "New Data Source" button to initiate the connection process.

3. In the modal that appears, select "Qdrant" as your data source.

4. Enter a name for your data source, input the credentials you obtained in the previous steps.

<img src="https://cdn.peaka.com/integrations/qdrant-1.png" />

5. Click "Authorize" to create your Qdrant data source.

Congratulations! You've successfully connected Qdrant to Upsolve. You can now access and manage your Qdrant data seamlessly within the Upsolve platform.
