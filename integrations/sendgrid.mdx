This article guides you through the process of connecting SendGrid to Upsolve, allowing you to seamlessly access your SendGrid data within the Upsolve platform.

## Prerequisites

Before you start, you will need an API Key from Open AI.

Follow the steps below to obtain it:

1. Go to this [link](https://www2.Sendgrid.com/app/account/integrations) to create or obtain your SendGrid API Key.

2. Click the "Create API Key" button to generate your API Key.

<img src="https://cdn.peaka.com/integrations/sendgrid-1.png" />

## Connecting SendGrid to Upsolve

Now that you have your requirements, you can connect it to Upsolve using the following steps:

1. Navigate to the data model page in Upsolve.

2. In the side menu bar, click the "New Data Source" button to initiate the connection process.

3. In the modal that appears, select "Sendgrid" as your data source.

4. Enter your data source name and API Key.

<img src="https://cdn.peaka.com/integrations/sendgrid-2.png" />

5. Click "Authorize" to create your SendGrid data source

Congratulations! You've successfully created a data source between SendGrid and Upsolve. You can now access and manage your SendGrid data seamlessly within the Upsolve platform.
