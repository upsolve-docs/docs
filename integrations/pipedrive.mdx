This article guides you through the process of connecting Pipedrive to Upsolve, allowing you to seamlessly access your Pipedrive data within the Upsolve platform.

## Prerequisites

Before you start, you will need an API Key from Pipedrive.

Follow the steps below to obtain it:

1. Visit the [Pipedrive website](https://app.pipedrive.com/).

2. It can be found under **Settings > Personal preferences > API**.

<img src="https://cdn.peaka.com/integrations/pipedrive-1.png" />

<Note>
  If you cannot find your API token here, it may be that the permission set you
  are in does not have that permission enabled.
</Note>

## Connecting Pipedrive to Upsolve

Now that you have your Pipedrive API Key, you can connect it to Upsolve using the following steps:

1. Navigate to the data model page in Upsolve.

2. In the side menu bar, click the "New Data Source" button to initiate the connection process.

3. In the modal that appears, select "Pipedrive" as your data source.

4. Enter a name for your data source name and paste the Pipedrive API Key into the "API Key" input field.

<img src="https://cdn.peaka.com/integrations/pipedrive-2.png" />

5. Click "Authorize" to create your Pipedrive data source.

Congratulations! You've successfully connected Pipedrive to Upsolve. You can now access and manage your Pipedrive data seamlessly within the Upsolve platform.
