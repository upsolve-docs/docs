This article guides you through the process of connecting Provet Cloud to Upsolve, allowing you to seamlessly access your Provet Cloud data within the Upsolve platform.

## Prerequisites

Before you start, you will need the following information from Provet Cloud.

- Client ID
- Client Secret
- Provet ID
- Environment

Follow the steps below to obtain it:

1. To obtain Client ID and Client Secret, check this [article](https://developers.provetcloud.com/restapi/authentication_oauth2.html#client-credentials-example-get-access-token).

2. To obtain Provet ID, check this [article](https://support.provet.cloud/hc/en-gb/articles/360009783098-Logging-In-to-Provet-Cloud).

3. To obtain Environment, check this [article](https://support.provet.cloud/hc/en-gb/articles/360010249038-Integrating-Third-party-Applications-with-Provet-Cloud).

## Connecting Provet Cloud to Upsolve

Now that you have obtained the necessary integration data and credentials, you can connect Provet Cloud to Upsolve using the following steps:

1. Navigate to the data model page in Upsolve.

2. In the side menu bar, click the "New Data Source" button to initiate the connection process.

3. In the modal that appears, select "Provet Cloud" as your data source.

4. Enter a name for your data source, input the credentials you obtained in the previous steps.

<img src="https://cdn.peaka.com/integrations/provet-cloud-1.png" />

5. Click "Authorize" to create your Provet Cloud data source.

Congratulations! You've successfully connected Provet Cloud to Upsolve. You can now access and manage your Provet Cloud data seamlessly within the Upsolve platform.
