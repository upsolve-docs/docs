This article guides you through the process of connecting MariaDB to Upsolve, allowing you to seamlessly access your MariaDB data within the Upsolve platform.

## Prerequisites

Before you start, you will need the following information from MariaDB.

- Host (Endpoint without the port or database name)
- Port (The port used to connect to your database)
- User (The username of the service account)
- Password (The password for the service account user)
- SSL, if active (Toggle "Connect Through TLS" switch)

## Connecting MariaDB to Upsolve

1. Navigate to the data model page in Upsolve.

2. In the side menu bar, click the "New Data Source" button to initiate the connection process.

3. In the modal that appears, select "MariaDB" as your data source.

4. Fill out the data source details form as shown in the image below, including:

   - Host (Endpoint without the port or database name)
   - Port (The port used to connect to your database)
   - User (The username of the service account)
   - Password (The password for the service account user)
   - Connect Through TLS (Toggle switch if SSL is active)

5. Click "Next" to establish the data source.

<img src="https://cdn.peaka.com/integrations/mariadb-1.png" />

Congratulations! You've successfully created a new data source between MariaDB and Upsolve. You can now access and manage your MariaDB data seamlessly within the Upsolve platform.
