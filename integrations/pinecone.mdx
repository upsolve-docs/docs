This article guides you through the process of connecting Pinecone to Upsolve, allowing you to seamlessly access your Pinecone data within the Upsolve platform.

## Prerequisites

Pinecone offers two distinct connection methods:

- Pinecone Connect
- API Key

Before you start, you will need the following information from Pinecone.

- API Key (if required for the connection method)

Follow the steps below to obtain it:

1. To obtain API Key, check this [article](https://docs.pinecone.io/guides/get-started/authentication).

## Connecting Pinecone to Upsolve

Now that you have obtained the necessary integration data and credentials, you can connect Pinecone to Upsolve using the following steps:

1. Navigate to the data model page in Upsolve.

2. In the side menu bar, click the "New Data Source" button to initiate the connection process.

3. In the modal that appears, select "ActiveCampaign" as your data source.

4. Choose the authentication method that best suits your needs.

5. If you're using the API Key connection method, simply enter your API Key and click the Authorize button.

6. If you're using the Pinecone Connect method, click the Connect to Pinecone button and allow Upsolve to access your Pinecone data.
