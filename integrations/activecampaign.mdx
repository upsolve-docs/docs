This article guides you through the process of connecting ActiveCampaign to Upsolve, allowing you to seamlessly access your ActiveCampaign data within the Upsolve platform.

## Prerequisites

Before you start, you will need an API Key and an API URL from ActiveCampaign.

Follow the steps below to obtain it:

1. Visit the [ActiveCampaign website](https://www.activecampaign.com/) and log in.

2. Click on the "Settings" button and click on the "Developer" button on the side bar, as shown in the image below:

<img src="https://cdn.peaka.com/integrations/active-campaign-1.png" />

3. Copy the API key and the API URL.

<img src="https://cdn.peaka.com/integrations/active-campaign-2.png" />

## Connecting ActiveCampaign to Upsolve

Now that you have your ActiveCampaign API URL and API Key, you can connect it to Upsolve using the following steps:

1. Navigate to the data model page in Upsolve.

2. In the side menu bar, click the "New Data Source" button to initiate the connection process.

3. In the modal that appears, select "ActiveCampaign" as your data source.

4. Enter a name for your data source name and paste the ActiveCampaign API URL and API Key into the input fields.

<img src="https://cdn.peaka.com/integrations/active-campaign-3.png" />

5. Click "Authorize" to create your ActiveCampaign data source.

Congratulations! You've successfully connected ActiveCampaign to Upsolve. You can now access and manage your ActiveCampaign data seamlessly within the Upsolve platform.
