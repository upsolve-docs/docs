This article guides you through the process of connecting Neo4J to Upsolve, allowing you to seamlessly access your Neo4J data within the Upsolve platform.

## Prerequisites

Before you start, you will need the following information from Neo4J.

- Host (Endpoint without the port or database name)
- Port (The port used to connect to your database)
- User Name (The username of the service account)
- Password (The password for the service account user)
- SSL, if active (Optional)
- Bolt Schema, if active (Optional)

Follow the steps below to obtain it:

1. To obtain the User Name and Password, go to the [article](https://support.neo4j.com/s/article/6787460237843-Neo4j-Aura-Database-Authentication-Neo4j-Database-Username-Password#:~:text=When%20you%20create%20an%20instance,note%20of%20the%20username%2C%20neo4j).

2. To obtain the Host and Port, go to the [article](https://neo4j.com/docs/operations-manual/current/configuration/ports/#_ports_used_by_neo4j).

## Connecting Neo4J to Upsolve

1. Navigate to the data model page in Upsolve.

2. In the side menu bar, click the "New Data Source" button to initiate the connection process.

3. In the modal that appears, select "Neo4J" as your data source.

4. Enter a name for your data source and input credentials you obtained in the previous step.

5. Click "Next" to establish the data source.

Congratulations! You've successfully connected Neo4J to Upsolve. You can now access and manage your Neo4J data seamlessly within the Upsolve platform.
