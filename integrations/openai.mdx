This article guides you through the process of connecting OpenAI to Upsolve, allowing you to seamlessly access your OpenAI data within the Upsolve platform.

## Prerequisites

Before you start, you will need an API Key from OpenAI.

Follow the steps below to obtain it:

1. Go to this [link](https://platform.openai.com/account/api-keys) to create or obtain your OpenAI API Key.

2. Click the "Create new secret key" button to generate your API Key.

<img src="https://cdn.peaka.com/integrations/openai-integration-1.png" />

## Connecting OpenAI to Upsolve

Now that you have your requirements, you can connect it to Upsolve using the following steps:

1. Navigate to the data model page in Upsolve.

2. In the side menu bar, click the "New Data Source" button to initiate the connection process.

3. In the modal that appears, select "OpenAI" as your data source.

4. Enter your data source name and API Key.

<img src="https://cdn.peaka.com/integrations/openai-integration-2.png" />

5. Click "Authorize" to create your OpenAI data source.

Congratulations! You've successfully created a new data source between OpenAI and Upsolve. You can now access and manage your OpenAI data seamlessly within the Upsolve platform.
