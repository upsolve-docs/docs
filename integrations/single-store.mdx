This article guides you through the process of connecting SingleStore to Upsolve, allowing you to seamlessly access your SingleStore data within the Upsolve platform.

## Prerequisites

Before you start, you will need the following information from SingleStore.

- User
- Password
- Host
- Port
- Database Name

Follow the steps below to obtain it:

1. Go to the SingleStore website and log in to your account if you're not already logged in.

2. Create a new workspace on SingleStore. Please note that this step may take some time.

3. Click the "Connect" button and select "Connect Directly." This will take you to the SingleStore dashboard.

<img src="https://cdn.peaka.com/integrations/single-store-1.png" />

4. Click on the "SQL IDE" tab. Here, you will find your user name, port, and host.

<img src="https://cdn.peaka.com/integrations/single-store-2.png" />

5. Select the "Access" tab. Here, you will find your password. You can reset your password if needed.

<img src="https://cdn.peaka.com/integrations/single-store-3.png" />

6. Select the "Database" tab and copy your database name.

## Connecting SingleStore to Upsolve

Now that you have your requirements, you can connect it to Upsolve using the following steps:

1. Navigate to the data model page in Upsolve.

2. In the side menu bar, click the "New Data Source" button to initiate the connection process.

3. In the modal that appears, select "SingleStore" as your data source

4. Enter your data source name, user, password, host, port, and database name. Optionally, enable the "Connect through TLS" switch if SSL is active.

<img src="https://cdn.peaka.com/integrations/single-store-4.png" />

5. Click "Authorize" to create your SingleStore data source.

<img src="https://cdn.peaka.com/integrations/single-store-5.png" />

Congratulations! You've successfully created a data source between SingleStore and Upsolve. You can now access and manage your SingleStore data seamlessly within the Upsolve platform.
