---
title: "How to Manage Your Partner API Key"
description: "This guide explains how to create, manage, and update Partner API Keys in Upsolve."
---

This guide will show you how to create, manage, and update your Partner API Key within the Upsolve platform.

When your partnership with Upsolve is active, the Developer section will appear in the sidebar on the homepage.

<img
  src="https://cdn.peaka.com/how-to-guides/how-to-manage-partner-api-key-1.png"
  alt="open developer page"
/>

### Creating a Partner API Key

1. Navigate to the [Developer](https://upsolve.studio/home/advanced-settings) section.
2. Click on the Partner API Key button to open a dialog box.
3. Fill in the form by entering your desired API Key Name.
4. Click on the Create button.

<img
  src="https://cdn.peaka.com/how-to-guides/how-to-manage-partner-api-key-2.png"
  alt="create partner api key dialog"
/>

Your Partner API Key will be displayed in the dialog box, where you can copy and use it.

<img
  src="https://cdn.peaka.com/how-to-guides/how-to-manage-partner-api-key-3.png"
  alt="created partner api key dialog"
/>

### Updating Status

By default, when a Partner API Key is created, its status is set to active. If you wish to deactivate the key, toggle the switch to the off position.

<img
  src="https://cdn.peaka.com/how-to-guides/how-to-manage-partner-api-key-4.png"
  alt="update status of partner api key"
/>

### Editing a Partner API Key

1. Click on the three dots icon of Partner API Key which you want to edit.

<img
  src="https://cdn.peaka.com/how-to-guides/how-to-manage-partner-api-key-5.png"
  alt="partner api key menu"
/>

2. Select the Edit Name button, which will open a dialog box.
3. Enter the new API Key Name and click on the Update button.

<img
  src="https://cdn.peaka.com/how-to-guides/how-to-manage-partner-api-key-6.png"
  alt="edit partner api key dialog"
/>

Your API Key name will be updated accordingly.

<img
  src="https://cdn.peaka.com/how-to-guides/how-to-manage-partner-api-key-7.png"
  alt="edited partner api key"
/>

### Deleting a Partner API Key

1. Click on the three dots icon of Partner API Key which you want to delete.

<img
  src="https://cdn.peaka.com/how-to-guides/how-to-manage-partner-api-key-5.png"
  alt="partner api key menu"
/>

2. Click the Delete button.

<img
  src="https://cdn.peaka.com/how-to-guides/how-to-manage-partner-api-key-8.png"
  alt="delete partner api key dialog"
/>

Your Partner API Key will be deleted.

<img
  src="https://cdn.peaka.com/how-to-guides/how-to-manage-partner-api-key-9.png"
  alt="deleted partner api key"
/>
