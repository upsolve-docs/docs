This article provides an introduction to the 'Records' node in Upsolve, explaining the various record-related nodes available, including 'Get a Record,' 'Get All Records,' 'Add a Record,' 'Update a Record,' and 'Delete a Record.'

<img
  src="https://cdn.peaka.com/flows/records-1.png"

/>

## Types of Records Nodes

In Upsolve, you can work with table data using the following 'Records' nodes:

<img
  src="https://cdn.peaka.com/flows/records-2.png"
  alt="Types of Records Nodes"
/>

### Get a Record

The **Get a Record** node allows you to retrieve a single record from your table. You can also filter the data using the "Filter | Visual SQL Editor" option.

<img
  src="https://cdn.peaka.com/flows/records-3.png"
  alt="Get a Record node parameters"
/>

### Get All Records

The **Get All Records** node enables you to fetch all records from your table. Like the 'Get a Record' node, you can filter the data using the "Filter | Visual SQL Editor" option.

<img
  src="https://cdn.peaka.com/flows/records-3.png"
  alt="Get All Records node parameters"
/>

### Add a Record

With the **Add a Record** node, you can insert new data into your table. You enter the data in JSON format within the "Edit Record Details" modal.

<img
  src="https://cdn.peaka.com/flows/records-4.png"
  alt="Add a Record node parameters"
/>

### Update a Record

The **Update a Record** node allows you to modify rows in your table. To do this, you need to select the table, specify the filter column and filter value, and then provide the new data in JSON format within the "Edit Record Details" modal.

<img
  src="https://cdn.peaka.com/flows/records-6.png"
  alt="Update a Record node parameters"
/>

### Delete a Record

Using the **Delete a Record** node, you can remove data from your table. To do this, select the table, specify the filter column, and provide the filter value.

<img
  src="https://cdn.peaka.com/flows/records-5.png"
  alt="Delete a Record node parameters"
/>

## Conclusion

These 'Records' nodes empower you to interact with table data efficiently, allowing you to retrieve, add, modify, and delete records as needed within your Upsolve applications.
