This article provides insights into reusable flows and explains how to utilize them effectively within your Upsolve workflows.

## What Are Reusable Flows?

Reusable flows are a versatile feature in Upsolve that allows you to create and reuse a specific set of actions and logic across different parts of your application. These flows can be employed within component flows, backend flows, or even other reusable flows, enhancing the modularity and reusability of your workflows.

## Creating a Reusable Flow

To create a reusable flow, follow these steps:

1. If you're not familiar with the process, refer to our guide on [How to Create a Flow](http://www.upsolve.com/docs/flows/how-to-create-a-flow/) for detailed instructions on creating flows.

2. Once you have your reusable flow ready, navigate to the flow where you want to implement it.

3. Click the plus icon in your flow editor to add a new node.

<img
  src="https://cdn.peaka.com/flows/reusable-flows-1.png"
  alt="using reusuable flow node"
/>

4. Select the 'Reusable Flow' button from the available node options.

5. From the configuration options, choose the specific reusable flow that you wish to implement.

<img
  src="https://cdn.peaka.com/flows/reusable-flows-2.png"
  alt="parameters of used reusable flow node"
/>

Congratulations! You have successfully implemented a reusable flow within your workflow.

## Conclusion

Utilizing reusable flows streamlines your development process, encourages code reusability, and simplifies maintenance, making it an essential tool for efficient application development in Upsolve.
