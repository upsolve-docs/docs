## Configuration

<ResponseField name="X-Axis" type="string or numeric" required>
  Category to group the line chart by
</ResponseField>

<ResponseField name="Y-Axis" type="numeric" required>
  Column to aggregate the line chart on. You could select multiple columns by **Add Series**
</ResponseField>

<ResponseField name="Segment by">
  Dividing series into meaningful subgroups. The data in each subgroups can then be aggregated via one of `Distinct count`, `Count`, `Sub`, `Average`, `Min`, `Max`
</ResponseField>
