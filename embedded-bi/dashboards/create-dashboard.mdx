Navigate to the **Build** application using the side navigation bar and click **Create dashboard** to create a new dashboard. There are two ways you can create dashboards:
1. [Point-and-click interface](/embedded-bi/dashboards/point-and-click-interface)
2. [Dash by Upsolve](/embedded-bi/dashboards/dash-by-upsolve)
