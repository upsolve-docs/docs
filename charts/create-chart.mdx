Navigate to the **Build** application using the side navigation bar and click **Create chart** to create a new chart. There are two ways you can create charts:
1. [Point-and-click interface](/charts/point-and-click-interface)
2. [Archie by Upsolve](/charts/archie-by-upsolve)
