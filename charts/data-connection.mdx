## Connect to your data sources

1. Navigate to the **Connect** application using the side navigation bar.
2. Select your data source type.
3. Input the corresponding credentials for the data source.
4. Click **Test connection** to see if connection is set up successfully. 
5. If the connection is successful, click **Compute metadata**.

![data-connection-setup](/images/data-connection/data-connection-setup.png)

You are now good to go to start [building charts](/concepts/chart-creation) and [creating dashboards](/concepts/dashboard-creation).
If you do not see the data source type you want to connect to in the **Connect** application, please raise a support ticket.
